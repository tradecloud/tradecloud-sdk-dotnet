# Add User

This example adds an identity to Tradecloud

## Prerequisites

Super powers

## Configure

In the source code:
- amend authenticationUrl
- fill in username
- fill in password
- add addIdentityUrl
- amend identity fields

## Run

```
➜  AddIdentity git:(master) ✗ dotnet run
```
